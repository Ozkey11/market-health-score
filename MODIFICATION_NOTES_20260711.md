# 改修内容（2026-07-11）

## 実装済み

1. スコア推移を2本線化
   - 紫: L2+L3スコア（左軸 0–100）
   - 橙: VIX / 日経VI（右軸・実数値）
   - 日付で突合し、休場日など一致しない日は線を無理につながない。

2. 表示期間
   - 1週間（5営業日）
   - 1ヶ月（22営業日）
   - 3ヶ月（66営業日）
   - 6ヶ月（132営業日）

3. ファンダメンタル環境
   - PC: 左に大底スコア、右に天井スコアを横並び。
   - スマホ: 同じ2列を中央揃えし、右ずれを防止。

4. お気に入り
   - 初期化処理が検索履歴保存関数の中に誤って入っていたスコープ不具合を修正。
   - 0件/1件でも画面遷移、並び替え、フィルタが動作。
   - 単独スキャン、一括スキャンを修正。
   - 分析画面の取得結果をお気に入り側へ自動同期。
   - 価格、前日比、総合、大底、天井、会社名を保存・表示。
   - 日本株の名称は Yahoo Finance の shortName/longName を取得後に更新。
   - 各行にチャートボタンを追加。

5. ローソク足
   - 3ヶ月 / 6ヶ月 / 1年切替。
   - OHLCと日付をキャッシュ対象へ追加。

6. Layer2表
   - スマホで列を非表示にせず横スクロール化。
   - 「解釈」列まで閲覧可能。

7. マーケット状態ラベル
   - 総合スコア、レジーム、反転確認、深度Dを別々に参照し、
     「全面パニック・大底探索」「大底圏・反転兆候」「回復初動」
     「上昇トレンド」「過熱警戒」「天井圏・反転兆候」などを表示。

8. エントリー準備度（試験実装）
   - 底の深さ 35%
   - 反転確認 25%
   - 短期安定化 20%
   - 市場心理 10%
   - データ自信度 10%
   - 総合スコアが中立〜天井側の場合は減衰補正。
   - 「待機 / 監視 / 準備 / 打診候補 / 反転確認済み」の5段階。
   - 総合スコアには混ぜず、独立した検証用指標として実装。

## 補足

依頼文にある「添付テキストファイル」はZIP内および同時アップロードファイルから確認できなかったため、マーケット状態ラベルとエントリー準備度は上記の暫定仕様で実装しています。仕様テキストが追加された場合は、ラベル名・閾値・構成比をその内容に合わせて置換できます。
